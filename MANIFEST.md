# Manifest

Определяем для каждого файла уровень готовности к миграции в Laravel.
Смотрим зависимости, стиль и связь с легаси — не только синтаксис.
Результат — шкала, а не «да/нет».

**Уровни:** `Legacy` → `LaravelReady` или `LegacyPerfect`; дальше `LaravelPerfect`. Два выхода из Legacy.

**Метки** (ставит разработчик): `legacy`, `adapter`, `for-legacy`.

Простота: одна команда, понятный вывод. Новые проверки — через правила, без правки ядра.

Не делаем: миграцию, генерацию кода, замену PHPStan/Psalm, автоматическое назначение меток, runtime-анализ.
